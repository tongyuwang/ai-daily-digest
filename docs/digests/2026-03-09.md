### 3. Compressing a Month of Research into 3 Hours with Claude
**[Post](https://x.com/heyrimsha/status/2030614728591167973)** · 👁️ 328K views · 🔖 9,906 bookmarks

A YC founder walkthrough on using Claude not as "a faster Google" but as a thinking partner. The key prompt sequence:
1. Feed it competitor pages, earnings transcripts, customer reviews, Reddit threads
2. Ask: *"What does every successful player in this market understand that their customers never say out loud?"*
3. Follow up: *"Show me the 3 assumptions this entire market is built on, and what would have to be true for each one to be wrong."*
4. Stress-test with: *"Write 5 questions a world-class investor would ask to destroy this idea, then answer each using only evidence in these documents."*

The tool didn't change. The questions did.
