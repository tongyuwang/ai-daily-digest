### 4. 💡 Claude Code Productivity Tips (SaaS Trenches Edition)
**[Tweet](https://x.com/i/web/status/2031457304328229184)**
👁 390,695 impressions · 🔖 7,016 bookmarks · ❤️ 2,923 likes

A viral thread from someone running a profitable SaaS. Actionable tips:
- **Shift-tab into planning mode** for any non-trivial feature; ask it to "do deep research on best practices" — plans survive context compaction far better than vibe-prompted work
- Install `mksglu/claude-context-mode` plugin (bigger context window via MCP references)
- Install `claude-warden` (protection against destructive actions)
- Build a `/documents/` folder: `platform-docs.md`, `ICPs.md`, `styleguide.md`, `roadmap.md`, `data-reference.md`
- Keep `CLAUDE.md` pointing at all of them
