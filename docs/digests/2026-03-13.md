### 2. 🔒 AWARE: Enterprise AI Agent Governance Framework
**[Tweet](https://x.com/i/web/status/2032103856549556697)**
👁 9,290 impressions · 🔖 144 bookmarks · ❤️ 105 likes

Glean's Work AI Institute (with Palo Alto Networks + Databricks) released the AWARE framework for governing AI agents in the enterprise — because the old security playbook was built for humans, not autonomous systems.

The five pillars:
- **Actor Intent** — Who/what is acting, and why?
- **Work Context** — Is this data sensitive in this context?
- **Autonomous Guardrails** — Is the agent within its declared scope?
- **Real-Time Risk Scoring** — How risky is this behavior right now?
- **Ecosystem Observability** — Can we trace what it did across every system?

Sobering stat: only 17% of orgs have automated controls for AI data flows. AI-specific breaches take 290 days to contain — 40% longer than traditional breaches.
