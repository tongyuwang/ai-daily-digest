## Key Themes

- **Harness engineering is the new craft** — The week's dominant narrative: it's not about the model, it's about the environment. From OpenAI's harness-ready repo prompt to "The Anatomy of an Agent Harness," the field is coalescing around systematic harness design as the actual differentiator.
- **Agent memory gets a real upgrade** — Hindsight beats every RAG system on LongMemEval using biomimetic memory (semantic + keyword + graph + temporal retrieval). Princeton drops research showing agents throw away the most valuable training signal they'll ever see — the next-state signal from every interaction.
- **Claude Code as the center of gravity** — An extraordinary number of bookmarks this week orbit Claude Code: 4-layer system architecture, 61-agent public repo, Anthropic hackathon winner open-sourcing everything, SaaS productivity tips going viral, and a full beginner masterclass. The ecosystem is maturing fast.
- **Self-improving agents going mainstream** — Autocontext (recursive self-improving harness), self-improving skills papers, and swarm agents in WASM sandboxes all hit in the same window. The shift from "AI tools" to "AI systems that improve themselves" is accelerating.
- **Enterprise AI governance gets a framework** — Glean, Palo Alto Networks, and Databricks co-develop AWARE: the first serious attempt to codify AI agent governance for enterprises. Only 17% of orgs have automated controls for AI data flows.
